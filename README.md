# ✨ LeetPost

### *Generate High-Engagement LeetCode Solutions Instantly* 🚀

**LeetPost** is a powerful Chrome extension designed for developers who want to share their LeetCode journey with style. It automatically extracts your code, analyzes your logic, and generates premium, "viral-ready" solution posts with AI — all within seconds.

---

## 🌟 Key Features

- **🔥 Viral Hook Titles:** Automatically generates catchy, emoji-rich titles designed for high engagement on LeetCode's "Solutions" tab.
- **⚡ Ultra-Fast Generation:** Powered by **Groq (Llama 3.3 70B)** and **Gemini 1.5 Flash**, delivering insights in milliseconds.
- **🧠 Built-in DSA Expertise:** Explains the "Aha!" moment and intuition behind every problem, making your posts educational and valuable.
- **💻 Native LeetCode Styling:** Uses `lang []` syntax to create beautiful side-by-side code tabs automatically.
- **🕒 Smart History & Cache:** Remembers your generated posts for each problem to save your time and API quota.
- **👤 Personalized Profile:** Automatically injects your LinkedIn, GitHub, and custom sign-offs into every post.

---

## 🛠️ Tech Stack

- **Core:** JavaScript (ES6+), Chrome Extension V3
- **AI Models:** Groq (Llama 3.3 70B Versatile), Google Gemini 1.5 Flash
- **Styling:** Vanilla CSS3 with premium Glassmorphism aesthetics

## 🚀 Getting Started

### 1. Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/aaravkashyap12/leetpost.git
   ```
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** (top right).
4. Click **Load unpacked** and select the `extension` folder from this project.

### 2. Configuration
1. Click the **LeetPost** icon in your toolbar.
2. Go to the **Settings** tab.
3. Choose your preferred provider (**Groq** is recommended for speed).
4. (Optional) Paste your own API key to increase limits. A shared key is included for immediate use!
5. Go to the **Profile** tab and set your name and social links.

### 3. Usage
1. Solve a problem on LeetCode.
2. Once your solution is **Accepted**, a "Generate Post" button will appear.
3. Click it, review your post, and hit **"Open & Paste"**.
4. Paste the content into the LeetCode Solution editor and you're done!

## 💡 Why LeetPost?

Writing a high-quality solution post for every problem I solved was becoming incredibly time-consuming. I wanted to share my logic and help others, but the manual effort was slowing down my learning. So, I built **LeetPost** to automate the process—turning technical code into engaging, educator-level content in a single click.
